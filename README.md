## NouMan Worker について

NouMan Worker は、農業現場を支える AI アシスタントです。

農家が日々の気づき・記録・判断をより自然に行えるよう、会話するだけで作業記録・気象情報・生育状況を管理できます。

**デザインの世界観:** 静かな信頼感・農業現場への敬意・Apple / MUJI のような引き算の美学
