## ブランドアイデンティティ（最重要）

NouMan は **「未来を予測するAI」ではありません。**

NouMan は **「観察を支えるAI」** です。

この違いがすべての広告の土台になる。

| 言ってよいこと | 言ってはいけないこと |
|-------------|-----------------|
| 「観察を、未来の豊かさに」 | 「AIが農業を変える」 |
| 「現場の違和感を受け止める」 | 「データで収益を最大化」 |
| 「話すだけで記録完了」 | 「高精度AIが判断」 |
| 「あなたの記憶が、畑を守る」 | 「スマート農業の新時代」 |
| 「農家と一緒に考える」 | 「農業DX」「IoT農業」 |

## レイアウト DNA

### 基本原則

- **コピーを最優先** — レイアウトはコピーに従う
- **左揃え** — 日本語は左からの視線フローに合わせる
- **余白を大きく** — 詰め込まない。余白は「呼吸」
- **情報量は最小** — 伝えることは1つ。多くて2つ
- **背景写真は世界観の根拠** — 農場・空・自然光・リアル
- **スマホは証拠** — コピーで言ったことをUIが証明する

### グリッド設計（App Store 縦型）

```
┌─────────────────────────────┐
│  [余白]                      │  ← 上部余白 10〜15%
│                              │
│  ヒーローコピー               │  ← 上部 20〜35%
│  （大・太・左揃え）            │
│                              │
│  サブコピー                  │  ← コピー直下
│  （中・通常・左揃え）          │
│                              │
│  ┌──────────────────┐       │
│  │   スマホ画面       │       │  ← 中下部
│  │  （実際のUI）      │       │
│  └──────────────────┘       │
│                              │
│  [CTA or ブランドメッセージ]  │  ← 下部
└─────────────────────────────┘
```

## 視線誘導 DNA

視線は「重力」に従う。上から下へ。左から右へ。

```
ロゴ（小・コーナー）
        ↓
コピー（大・インパクト）
        ↓
サブコピー（補足）
        ↓
CTA（行動喚起）
        ↓
スマホ（証拠）
        ↓
ブランドメッセージ（余韻）
```

### 視線が止まるポイント

1. **コントラスト** — 白背景 × 黒テキスト OR 暗背景 × 白テキスト
2. **サイズの差** — ヒーローコピーとサブコピーのサイズ比は 3:1 以上
3. **余白** — テキストの周囲に十分な余白があると視線が集中する

## コピー DNA

### 原則

- **短く** — 1行 10〜16文字以内が理想
- **強く** — 動詞で終わる or 体言止め
- **一瞬で理解できる** — 農家が読まなくてもわかる

### 良いコピーの構造

```
[課題の共感] + [解決の予感]
または
[現状の否定] + [新しい未来]
または
[驚き] + [証拠]
```

### NouMan コピー例

| パターン | コピー例 |
|---------|---------|
| 課題共感 | 「なんか葉っぱがおかしい」── その違和感が、答えだった。 |
| 現状否定 | もう、泥だらけの手で入力するのは終わりにしよう。 |
| 未来提示 | 観察を、未来の豊かさに。 |
| 驚き | ただのつぶやきが、自動で記録になる。 |
| 先回り | 天候と記憶から、次の「想定外」を先回り。 |

## 背景写真 DNA

### 使ってよいもの

| カテゴリ | 具体例 |
|---------|-------|
| 農地 | 整然とした畑の畝・マルチシート・ハウス |
| 空 | 青空・雲・夕暮れ・夜明け |
| 自然光 | 朝の光が差し込む農場・逆光の葉 |
| 農家 | 作業中の手元・後ろ姿・農機具（人の顔は避ける） |
| 作物 | 新鮮な野菜・果実・生育中の苗 |

### 使ってはいけないもの

| 禁止 | 理由 |
|------|------|
| CG・3D レンダリング | リアリティがなく信頼感が損なわれる |
| ストックフォトのステレオタイプ農家 | 白衣・満面の笑顔・作り物感 |
| 都会的な風景 | NouMan の世界観と合わない |
| 暗すぎる・汚い農場 | ブランドイメージが損なわれる |
| テキストが多く入った写真 | コピーと競合する |

## ブランドカラーの使い方 DNA

| 色 | Hex | 広告での役割 |
|----|-----|------------|
| Brand Green | `#4C8C5A` | CTA・強調テキスト・アクセントのみ（5〜10%） |
| White | `#FFFFFF` | コピーテキスト・背景（暗背景の場合） |
| Black / Deep | `#000000` / `#1E3A2F` | コピーテキスト（明背景の場合） |
| Beige | `#EAD9B5` | 温かみが必要な背景・セカンダリ |

**Brand Green を背景として大きく使わない。**
緑は農業の色でもあるが、使いすぎると「ただの農業アプリ」になる。

## 禁止パターン DNA

以下は「悪いクリエイティブ」の典型。毎回確認すること。

| 禁止 | 理由 |
|------|------|
| 機能の羅列 | 読まれない・ダウンロードされない |
| 情報量が多い | 視線が迷う・何も伝わらない |
| 複数の CTA | 行動できない |
| ロゴを大きくセンターに | 広告ではなく名刺になる |
| グラデーション | トレンドに見え、時代遅れになる |
| テキストに色を多用 | ブランドが安定しない |
| スマホを中心に大きく | UI のデモ動画になる。広告ではない |
| 英語の内部概念名 | 農家に伝わらない |
