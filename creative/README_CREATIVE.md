## Submission Policy ── 納品ルール

NouMan では **「途中経過」をユーザーへ提出しません。**

| スコア | 行動 |
|-------|------|
| **95点以上** | ユーザーへ提出する ✅ |
| 90〜94点 | 再制作してから提出する |
| 89点以下 | 設計から見直して再制作する |
| Anti-Pattern 該当 | 採点せず即再制作する |

**品質 ＞ 速度**

```
30分かかっても 95点
5分で 82点
→ 95点を選択する
```

**1枚目が 95点に達するまで、2枚目以降を制作しない。**

再制作の際は「なぜ低いか」を特定してから修正すること。「なんとなく修正」は禁止。
