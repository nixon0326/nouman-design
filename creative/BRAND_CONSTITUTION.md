## ブランドの思想

```
会話と観察から農場の"記憶"を育て、
AIが最適な判断をサポートするアプリ。
```

NouManは「農業DXツール」ではない。
農家の経験・観察・感覚を「記憶」として蓄積し、
AIがそれを活かして次の判断を支援するプラットフォームである。

**タグライン:** 観察を、未来の豊かさに。

---

## ブランドの6つのコア概念

これらはデザインのすべての場面に表れなければならない。

| 概念 | 読み | 意味 |
|------|------|------|
| **観察** | Observation | 農場の細部を見逃さない。変化に気づくことが始まり |
| **記憶** | Memory | 観察を蓄積し、過去と未来をつなぐ資産にする |
| **未来供給** | Future Supply | 今日の記録が、明日の判断精度を高める |
| **会話** | Conversation | AIとの自然な対話で、記録の壁をゼロにする |
| **つながり** | Connection | 農場・人・AIが有機的につながる |
| **気づき** | Insight | データではなく、意味のある発見をもたらす |

---

## ロゴの意味

ロゴの各要素には意図がある。**描き直し・改変・再解釈を禁止する。**

| 要素 | 意味 |
|------|------|
| **芽（メインシンボル）** | 成長・可能性・未来供給。農業と未来の両立 |
| **吹き出し（右上）** | AIとの自然な会話。テキスト入力ではなく、話しかける体験 |
| **圃場ライン（下部）** | 農場・大地・つながり。足元にある現実への敬意 |
| **3つの点** | 観察・記録・気づきの積み重ね。継続が価値になる |

> ロゴは「記録アプリ」ではなく「農場の記憶を育てるAI」を象徴している。

## ブランドカラー

**正規のカラーパレット（5色体系）:**

| 役割 | カラー名 | HEX | 使用用途 |
|------|---------|-----|---------|
| **Deep Forest** | ダークグリーン | `#1E3A2F` | 最重要テキスト・見出し・最高の信頼感を伝える場面 |
| **Brand Green** | ブランドグリーン | `#4C8C5A` | メインアクセント・CTAボタン・アイコン。**画面の5〜10%のみ** |
| **Light Green** | ライトグリーン | `#A8C59A` | 背景・セカンダリ要素・余白感の演出 |
| **Warm Beige** | ウォームベージュ | `#EAD9B5` | 温かみ・自然感・BEFORE/AFTER対比など |
| **Near Black** | テキストブラック | `#222222` | 本文テキスト・説明文。純粋な#000000は使わない |

**重要:** テキストカラーは `#222222`（Near Black）を使用する。`#000000`（Pure Black）は使用しない。

### 背景カラーの使い分け

| 背景 | 使用場面 |
|------|---------|
| `#FFFFFF` White | デフォルト。清潔感・余白重視の場面 |
| `#A8C59A` Light Green | Feature Poster（Format A）の主背景 |
| `#EAD9B5` Warm Beige | BEFORE/AFTER比較・温かみが必要な場面 |

## ブランドトーン

NouManのコミュニケーションはこの3軸で構成される：

```
自然で、やさしく、信頼できる
テクノロジーを感じさせつつ、あたたかみがある
未来志向だが、地に足がついている
```

### トーン別の言葉の使い方

| 場面 | 使う言葉 | 使わない言葉 |
|------|---------|------------|
| コピー | 「話すだけで」「観察が」「記録が育つ」 | 「最先端AI」「革命的」「劇的に変わる」 |
| 機能説明 | 「自動で」「そのまま」「手間なく」 | 「高度な」「スマートに」「パワフルな」 |
| CTA | 「無料で始める」「今すぐ試す」 | 「今すぐ登録！」「見逃すな！」 |

## アイコン思想

5つのコンセプト案から選ばれた方向性：

| # | コンセプト | 特徴 |
|---|----------|------|
| **#1 メインコンセプト** | 採用済みメインロゴ。芽+吹き出し+圃場ライン | 現行の正式アイコン |
| #2 N×芽×圃場 | "N"の字と農業シンボルの融合 | ブランドレター軸 |
| #3 観察の波紋 | 観察・気づきの広がりを表現 | 概念的・抽象的 |
| #4 会話×葉 | 対話と自然の融合 | 温かみ重視 |
| #5 シンプルN | 最小限のレターマーク | モノクロ親和性高 |

> **現行アイコンは #1 メインコンセプト。** アイコンの描き直し・改変は禁止。

## ブランドが表現すべき世界観

NouManのビジュアルは「農業現場への敬意と静かな信頼感」を体現する。

- 大げさなビジュアルエフェクトを使わない
- 農場の写真は「主役」ではなく「コピーの額縁」として機能させる
- 白い余白は贅沢ではなく、信頼の表現
- グリーンは主張しない。存在することで伝わる
- テクノロジーの存在は「自然に」感じさせる

---

## 禁止事項

ブランド価値を毀損するデザインパターン：

| 禁止 | 理由 |
|------|------|
| Pure Black `#000000` の使用 | Near Black `#222222` が正規 |
| 写真を背景として前面に出す | 写真はコピーの額縁。主役はコピー |
| 鮮やかすぎるグリーン（#4C8C5A 以外） | ブランドカラーの逸脱 |
| アニメーション的・派手なグラデーション | 農業現場への敬意と相容れない |
| ロゴの改変・描き直し | 元データのみ使用 |
