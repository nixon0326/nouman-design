# NouMan Creative Review
## 制作後自己採点システム

> すべての制作物の完成後、このシートで採点すること。
> **90点未満は再制作する。** 例外なし。
> 採点は「自分が農家だったら」という視点で行う。

---

## 採点シート

制作物を完成させたら、以下の8項目を採点してください。

---

### ① 3秒で止まるか　/ 20点

App Store をスクロールしている農家の指が、3秒以内に止まるか。

| 点数 | 基準 |
|------|------|
| 18〜20点 | 確実に止まる。何かが引っかかる |
| 14〜17点 | 止まるかもしれない |
| 10〜13点 | 通り過ぎる可能性が高い |
| 0〜9点 | 確実に通り過ぎる |

**チェック項目:**
- [ ] ヒーローコピーが一瞬で目に入るか
- [ ] 背景写真に「引き」があるか
- [ ] ひと目見て「自分ごと」と感じるか

---

### ② コピー　/ 15点

ヒーローコピーが短く・強く・一瞬で理解できるか。

| 点数 | 基準 |
|------|------|
| 13〜15点 | 短い・強い・農家に刺さる |
| 10〜12点 | 悪くはないが印象に残らない |
| 6〜9点 | 長い・わかりにくい・機能説明になっている |
| 0〜5点 | 読まれない |

**チェック項目:**
- [ ] ヒーローコピーが 16文字以内か
- [ ] 機能説明になっていないか
- [ ] 感情が動くか（共感・驚き・希望のどれか）
- [ ] 農家の言葉で書かれているか（専門用語・英語なし）

---

### ③ 視線誘導　/ 15点

ロゴ → コピー → CTA → スマホ の順番で視線が流れるか。

| 点数 | 基準 |
|------|------|
| 13〜15点 | 自然に視線が流れる |
| 10〜12点 | 概ね流れるが一部迷う |
| 6〜9点 | 視線が迷う・止まる |
| 0〜5点 | どこを見ればよいか不明 |

**チェック項目:**
- [ ] 最初に目が行くのがコピーか（ロゴではない）
- [ ] 次にサブコピーか CTA に自然に目が移るか
- [ ] スマホが「最後の証拠」として機能しているか

---

### ④ 余白　/ 10点

「置いていないもの」が広告の価値を高めているか。

| 点数 | 基準 |
|------|------|
| 9〜10点 | 余白が「呼吸」として機能している |
| 7〜8点 | 余白はあるが効果的でない |
| 4〜6点 | 詰め込みすぎ |
| 0〜3点 | 情報過多・読めない |

**チェック項目:**
- [ ] テキスト周囲に十分な余白があるか
- [ ] 1画面に伝えることが 1〜2 に絞られているか
- [ ] 背景に「空間」があるか

---

### ⑤ ブランド整合性　/ 10点

NouMan の世界観・アイデンティティが伝わるか。

| 点数 | 基準 |
|------|------|
| 9〜10点 | 「NouManだ」と一瞬でわかる |
| 7〜8点 | 悪くはないが NouMan でなくてもよい |
| 4〜6点 | 別のアプリにも見える |
| 0〜3点 | ブランドが全く伝わらない |

**チェック項目:**
- [ ] Brand Green `#4C8C5A` が適切に使われているか（5〜10%）
- [ ] ロゴが小さく・コーナーに配置されているか
- [ ] NouMan のメッセージ（「観察を支えるAI」）の空気感があるか
- [ ] Apple/MUJI 的な「静かな信頼感」があるか

---

### ⑥ スマホ配置　/ 10点

スマホが「証拠」として正しく機能しているか。

| 点数 | 基準 |
|------|------|
| 9〜10点 | コピーを視覚的に証明している |
| 7〜8点 | 配置されているが効果的でない |
| 4〜6点 | スマホが主役になっている |
| 0〜3点 | UIが改変されている・不自然 |

**チェック項目:**
- [ ] 実際のアプリ画面を使用しているか
- [ ] UI が改変されていないか
- [ ] スマホの配置がコピーの「証拠」になっているか
- [ ] スマホが広告の主役になっていないか

---

### ⑦ 背景写真　/ 10点

背景が「NouMan の世界観」を表現しているか。

| 点数 | 基準 |
|------|------|
| 9〜10点 | 農業現場のリアルな空気感がある |
| 7〜8点 | 悪くはないが個性がない |
| 4〜6点 | 農業と関係が薄い・ストックフォト感 |
| 0〜3点 | CG・不自然・ブランドに合わない |

**チェック項目:**
- [ ] 実写の農場・空・自然光か
- [ ] CG・3Dレンダリングではないか
- [ ] コピーテキストの可読性を妨げていないか
- [ ] 「農業現場への敬意」が感じられるか

---

### ⑧ CTA　/ 10点

行動喚起が明確で、1つだけか。

| 点数 | 基準 |
|------|------|
| 9〜10点 | シンプル・明確・1つだけ |
| 7〜8点 | 存在するが目立たない |
| 4〜6点 | 複数ある・わかりにくい |
| 0〜3点 | ない・逆効果 |

**チェック項目:**
- [ ] CTA が1つだけか
- [ ] 「無料でためす」「ダウンロード」など行動が明確か
- [ ] 過剰な装飾をしていないか

## 判定

| 点数 | 判定 | アクション |
|------|------|----------|
| 95〜100 | EXCELLENT | そのまま提出 |
| 90〜94 | PASS | 提出可。改善点をメモ |
| 80〜89 | REWORK | 最低スコアの項目を修正して再採点 |
| 70〜79 | MAJOR REWORK | 設計から見直す（Design Flow ① から再開） |
| 0〜69 | REJECT | 作り直す |

## Anti-Pattern チェック（採点前に必ず確認）

採点の前にまず、以下の「即アウト」項目を確認すること。  
1つでも該当する場合は採点せず即再制作する。

| # | Anti-Pattern | 確認 |
|---|-------------|------|
| AP-01 | 背景写真がコピーより目立っている | ☐ |
| AP-02 | スマホが画面の 50% 未満のサイズ | ☐ |
| AP-03 | スマホ内の UI テキストが読めないサイズ | ☐ |
| AP-04 | コピーエリアが画面の 30% 未満 | ☐ |
| AP-05 | 1枚に 3 つ以上のメッセージがある | ☐ |
| AP-06 | 「読ませる」前提で設計している（LP 化） | ☐ |
| AP-07 | 感情が動かない（機能説明のみ） | ☐ |

**これらは「0点要素」ではなく「再制作確定」要素。**

---

## 再制作時の確認

再制作をする前に、以下を確認する：

1. **Anti-Pattern に該当しているか？** → 該当箇所が根本原因
2. **使っているフォーマットは正しいか？** → Format A（Feature Poster）/ Format B（App Screenshot）
3. **コピーは正しいか？** → コピーが間違っていると他は全部崩れる
4. **Design Flow の順番通りに作ったか？** → レイアウトから入ると失敗する
